Distributed LLM
Inference Engine

Distributed inference cluster for DeepSeek R1 and Llama 3.2B using model parallelism, increasing throughput 2–4×.

C++ Python CUDA Distributed Computing
2–4×
THROUGHPUT
prompt → tensor ∥ pipeline split → CUDA compute/comm overlap → 2–4× tokens/s
Parallelism Strategy
Tensor parallelism splits individual weight matrices across nodes; pipeline parallelism stages micro-batches to keep all GPUs saturated. CUDA kernels overlap inter-node communication with compute.
Cluster & Results
Implemented in C++ with Python bindings for both DeepSeek R1 and Llama 3.2B. The 2–4× throughput gain over single-GPU baselines scales with model size — larger models see the higher end of the range.
